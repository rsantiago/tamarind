# 20 Theme Concepts for Tamarind

We can emulate the aesthetics of these iconic platforms to offer users widely varied design languages.

## Social / Feed Style
1.  **"Gram" (Instagram)**: Mobile-first, card-based layout, heavy focus on imagery, square aspect ratios, minimal text chrome, infinite scroll feel.
2.  **"Blue" (Facebook Classic)**: Clean 3-column layout, blue navbar, white cards on light grey background, dense information hierarchy.
3.  **"Bird" (Twitter/X)**: Single focused feed column, prominent sidebars for navigation and "trends" (tags), distinctive typography (Chirp/Helvetica), high contrast borders.
4.  **"Pin" (Pinterest)**: Masonry grid layout, varying card heights, image-dominant, rounded corners, ideal for visual portfolios.
5.  **"Book" (Notion)**: Extremely minimal, emoji icons for pages, monochromatic, sans-serif, focuses on structure and hierarchy.

## Entertainment / Media
6.  **"Stream" (Netflix)**: Dark mode default, "Hero" banner with video/image background, horizontal scrolling lists (carousels) for collections.
7.  **"Tube" (YouTube)**: Sidebar navigation, grid of thumbnails with metadata (views/date) below, clean white/black theme toggle.
8.  **"Pod" (Spotify)**: Dark/green aesthetic, list layouts that look like playlists, sticky player (or header) at bottom, sleek gradients.
9.  **"News" (NYTimes)**: Serif headlines, multi-column newspaper layout, black & white with sparse red accents, dense typography.
10. **"Mag" (The Verge)**: Bold, neon accents, distinct retro-futuristic fonts, large chaotic grids ("Bento box" style).

## Tech / Developer
11. **"Doc" (GitBook)**: Left sidebar tree navigation, right sidebar Table of Contents, clean sans-serif, light background, code-block focus.
12. **"Hub" (GitHub)**: Terminator font for code, utilitarian grey borders, README-style rendering, "tabs" for navigation.
13. **"Overflow" (StackOverflow)**: Utility-focused, orange accents, voting-style layout for content relevance, dense lists.
14. **"Term" (Terminal)**: Green text on black background, monospace fonts everywhere, blinking cursor effects, no images.
15. **"Hacker" (Hacker News)**: Beige background, tiny font, density maximum, orange header bar, purely functional.

## Design / Aesthetic
16. **"Swiss" (Existing Default)**: High contrast, large typography, grid systems, whitespace.
17. **"Brutal" (Brutalism)**: Clashing colors, raw HTML borders, web-safe fonts (Times New Roman, Courier), intentionally unpolished.
18. **"Glass" (MacOS/iOS)**: Heavy usage of backdrop-filter (blur), translucency, rounded corners, soft shadows, pastel gradients.
19. **"Paper" (Medium)**: Center-aligned, excessive whitespace, serif body font, focus on "reading mode," no distractions.
20. **"Retro" (Windows 95)**: Beveled grey buttons, windows with "X" close icons, pixel fonts, "Start" menu navigation.
