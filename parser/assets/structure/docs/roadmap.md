# The Future of Tamarind

We have reached version 2.0, but the journey continues.

## Q1 2026 Objectives

*   **RSS / Atom Feeds**: Essential for the open web. We want Tamarind to automatically generate XML feeds for articles.
*   **Sitemap Generation**: To help search engines index your content effectively.
*   **Tag Cloud**: A dedicated page to browse articles by tag.

## Immediate Priorities (Developer Shortcodes)

*   **Mermaid.js Diagrams**: Diagrams as code support.
*   **Callouts/Admonitions**: Styled blocks for warnings, info, tips.
*   **Terminal/Code Windows**: Styled CLI output blocks.
*   **Code File Include**: Embed actual code files from source.
*   **GitHub Gists**: Embed Gists.
*   **Math Support**: LaTeX/KaTeX rendering.

## Q1 2026 Objectives

*   **Live Reload**: Using WebSockets to refresh the browser instantly when you save a markdown file.
*   **Themes**: A command like `tamarind init --theme=dark-journal` to choose between different aesthetic starting points.

## Completed
*   ✅ **Dynamic Menu**: Pages now automatically populate the navigation.
*   ✅ **Embedded Assets**: The binary is fully self-contained.
*   ✅ **Clean Architecture**: Refactored logic into a maintainable state.
