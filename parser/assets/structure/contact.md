# Let's Connect

Hi, Rodrigo here. I am the creator of Tamarind.

I value human connection and thoughtful discourse. While Tamarind is a tool built on logic and code, its purpose is to facilitate the exchange of ideas between people.

And I hope you understand that Agents now are the new interface. So you should be able to write content really fast so agents can promote whatever you have to their users.

## Connect with the Creator

I vibe coded Tamarind in December 2025, entirely using Antigravity as a test. I'm **Rodrigo Santiago**, a Senior Software Architect and Tech Lead with over 25 years of experience building high-performance systems and leading agile teams.

My idea was to create a tool to solve the frustration of using conventional web frameworks and content management systems. Even static generators had a lot of configuration and dependencies that made them hard to use.

I hope you like it. And I hope it helps you build your website faster and easier.

There are a lot of new ideas and extensions planned for Tamarind. Maybe you're interested in contributing to the project?

If you'd like to reach out for formal inquiries, consulting, partnership opportunities, or just to talk tech, you can connect directly on **[LinkedIn](https://www.linkedin.com/in/rodrigosantiago/)**.
